Prueba Git de David Martínez
============================

Paráctica de manejo del control de versiones Egit para Eclipse y subida de repositorio a GitHub

Instalación
-----------

Se necesita instalar lo siguiente:

*	Java SDK
*	Eclipse Indigo
*	Plugin Egit
*	Repositorio en [GitHub](http//:www.github.com)
